                        AEGIS TUI — QUICK REFERENCE
════════════════════════════════════════════════════════════════════════════

┌─ LAUNCH ────────────────────────────────────────────────────────────┐
│  aegis wakeup (w)     Interactive menu                              │
│  aegis dashboard (dash)  Agent dashboard                            │
│  aegis chat (c)       AI chat                                       │
│  bun run index.ts <cmd>  Via bun                                    │
└──────────────────────────────────────────────────────────────────────┘

┌─ DASHBOARD KEYBOARD ────────────────────────────────────────────────┐
│  Tab         Cycle focus: log -> agents -> command                  │
│  Up / Down   Scroll log / Recall command history                    │
│  PgUp / PgDn Scroll log by 10 lines                                 │
│  Enter       Execute command                                        │
│  Backspace   Delete character                                       │
│  Ctrl+Q/C    Quit dashboard                                         │
└──────────────────────────────────────────────────────────────────────┘

┌─ DASHBOARD COMMANDS ────────────────────────────────────────────────┐
│  spawn <name> (s)        Launch an agent                            │
│  spawn --type <type>     Launch with agent type                     │
│  spawn --script <path>   Launch with custom script                  │
│  spawn --tag <tag>       Launch with tags (repeatable)              │
│  kill <name> (k)         Stop an agent                              │
│  kill all                Stop all agents                            │
│  list (ls)               Show all agents (status, pid, uptime)      │
│  status (st)             Show system info (version, mem, uptime)    │
│  help (h)                Show commands                              │
└──────────────────────────────────────────────────────────────────────┘

┌─ CHAT KEYBOARD ─────────────────────────────────────────────────────┐
│  Enter       Send message                                           │
│  Alt+Enter   Insert newline (multiline input)                       │
│  Up / Down   Message history / Multiline navigation                 │
│  Left / Right  Move cursor                                          │
│  Home / End  Jump to line start / end                               │
│  PgUp / PgDn Scroll message history                                 │
│  Esc         Cancel streaming / Clear input                         │
│  Backspace   Delete character                                       │
│  Ctrl+Q/C    Quit chat                                              │
└──────────────────────────────────────────────────────────────────────┘

┌─ DASHBOARD LAYOUT ──────────────────────────────────────────────────┐
│                                                                      │
│  ╭─ AEGIS DASHBOARD ──── v0.1.0  Ctrl+Q Quit ╮                     │
│  │ AGENTS           │ ACTIVITY LOG           │                      │
│  │ ● worker         │ ✓ Agent spawned        │                      │
│  │ · main           │ → Agent exited         │                      │
│  │                  │ ✕ Error                │                      │
│  ╰─ ● MEM:7% · CPU:0% · SESS:1 · UP:42m ──╯                      │
│  $ command here                                                      │
│                                                                      │
└──────────────────────────────────────────────────────────────────────┘

┌─ CHAT LAYOUT ───────────────────────────────────────────────────────┐
│                                                                      │
│  ╭─ AEGIS CHAT ────────── v0.1.0  Ctrl+Q Quit ╮                    │
│  → You  Ask a question...                                            │
│  → Aegis  Response streaming here...                                 │
│  │ text█                                                              │
│  · Enter to send | Alt+Enter newline | →→ history | Ctrl+Q quit     │
│                                                                      │
└──────────────────────────────────────────────────────────────────────┘

┌─ AGENT STATUS INDICATORS ───────────────────────────────────────────┐
│  ● green   Running / busy                                           │
│  ● yellow  Idle (no active task)                                    │
│  · muted   Stopped                                                  │
│  ✕ red     Error                                                    │
└──────────────────────────────────────────────────────────────────────┘

┌─ ACTIVITY LOG SYMBOLS ──────────────────────────────────────────────┐
│  ·        Info message                                              │
│  ✓        Success (spawned, recovered)                              │
│  →        Event (exited, state changed)                             │
│  !        Warning (recovery, issues)                                │
│  ✕        Error (failure, crash)                                    │
└──────────────────────────────────────────────────────────────────────┘

┌─ CHAT SYMBOLS ──────────────────────────────────────────────────────┐
│  → You    Your message                                              │
│  → Aegis  AI response                                               │
│  ·        System message                                            │
│  █        Cursor position                                           │
│  Streaming...  Response in progress                                │
└──────────────────────────────────────────────────────────────────────┘

┌─ REQUIREMENTS ──────────────────────────────────────────────────────┐
│  Terminal >= 80x24  |  TTY (no pipes)  |  ANSI escapes             │
│  Chat: ANTHROPIC_API_KEY env var required                           │
└──────────────────────────────────────────────────────────────────────┘

┌─ TROUBLESHOOTING ───────────────────────────────────────────────────┐
